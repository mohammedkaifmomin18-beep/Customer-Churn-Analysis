# 📊 Customer Churn Analysis | Data Analytics Project

## 📌 Problem Statement

Analyze customer churn behavior to identify why customers leave the telecom company and provide business recommendations to reduce churn.

---

# 🎯 Objectives

- Identify major churn drivers
- Analyze customer behavior patterns
- Discover high-risk customer groups
- Build an interactive Power BI dashboard
- Provide business recommendations

---

# 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- PostgreSQL
- SQL
- Power BI
- GitHub

---

# 📂 Dataset

Dataset contains telecom customer information including:

- Customer demographics
- Internet services
- Contract type
- Payment methods
- Monthly charges
- Churn status

---

# 🧹 Data Cleaning (Python)

Performed:

- Missing value treatment
- Converted TotalCharges to numeric
- Removed invalid records
- Data type corrections

EDA was performed using Python to analyze:

- Churn distribution

- Payment method analysis
- Customer behavior patterns

---

# 🗄️ SQL Analysis

SQL queries were used to analyze:

- Churn rate

- Customer segmentation

# 🔍 Key Insights

- Month-to-month customers show highest churn risk
- Electronic check users churn more frequently
- Fiber optic users show higher churn tendency
- Customers with higher monthly charges are more likely to churn
- Long-term contract customers are more loyal

---

# 💡 Business Recommendations

- Promote long-term contracts using discounts
- Improve customer support for fiber optic users
- Target high-risk customers with retention campaigns
- Provide incentives for automatic payment methods
